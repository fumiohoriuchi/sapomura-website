# サポ村.com - サポーター村を作る会 公式ウェブサイト

サポーターが作るサポーターのためのサポーターによるツアー

## 🚀 開発環境のセットアップ

### 必要な環境
- Node.js 18.x 以上
- npm または yarn

### インストール

```bash
npm install
# または
yarn install
```

### 開発サーバーの起動

```bash
npm run dev
# または
yarn dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いて確認できます。

## 📦 ビルド

```bash
npm run build
# または
yarn build
```

## 🌐 デプロイ

このプロジェクトはVercelでホスティングされています。

### Vercelへのデプロイ手順

1. GitHubにリポジトリをプッシュ
2. Vercelで新規プロジェクトを作成
3. GitHubリポジトリを連携
4. 自動デプロイが開始されます

## 📝 技術スタック

- **フレームワーク**: Next.js 14 (App Router)
- **スタイリング**: Tailwind CSS
- **言語**: TypeScript
- **ホスティング**: Vercel
- **CMS**: Sanity (予定)

## 📄 ページ構成

- `/` - トップページ
- `/about` - サポ村について
- `/achievements` - 過去の実績
- `/schedule` - 現在の日程
- `/register` - 会員登録
- `/contact` - お問い合わせ

## 🎨 デザインコンセプト

- 日本代表サポーターをイメージした青(サムライブルー)をテーマカラーに使用
- シンプルで親しみやすいデザイン
- レスポンシブ対応(スマートフォン・タブレット・PC)
